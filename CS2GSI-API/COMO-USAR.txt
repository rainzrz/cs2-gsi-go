CS2 GSI API - COMO USAR
========================

PASSO A PASSO NO POWERSHELL:

1. Abrir PowerShell como Administrador
   - Pressione Win + X
   - Selecione "Windows PowerShell (Admin)"

2. Navegar para a pasta CS2GSI-API:
   cd caminho\para\cs2-gsi-go\CS2GSI-API
   
   Exemplo:
   cd C:\Users\SeuUsuario\Desktop\cs2-gsi-go\CS2GSI-API
   ou
   cd D:\Projetos\cs2-gsi-go\CS2GSI-API

3. Executar o script (escolha uma opcao):

   OPCAO A - Script simples (RECOMENDADO):
   .\start-simple.ps1

   OPCAO B - Script completo:
   .\start-all.ps1

   OPCAO C - Manual:
   dotnet run
   (em outro terminal: python -m http.server 8080)

URLS DISPONIVEIS:
================

API Swagger (Documentacao):
http://localhost:5000/swagger

Interface HTML (Visual):
http://localhost:8080

Health Check:
http://localhost:5000/api/health

PROXIMOS PASSOS:
================

1. Abra http://localhost:8080 no navegador
2. Clique em "Verificar Status"
3. Teste os endpoints da API
4. Certifique-se de que o CS2 esta rodando com GSI

PARA PARAR:
===========

Pressione Ctrl+C no terminal

PROBLEMAS COMUNS:
=================

- Erro de permissao: Execute como Administrador
- Porta ocupada: Feche outros programas usando as portas 5000/8080
- NET nao encontrado: Baixe em https://dotnet.microsoft.com/download
- Python nao encontrado: Baixe em https://www.python.org/downloads/
- Pasta errada: Certifique-se de estar na pasta que contem Program.cs

ARQUIVOS CRIADOS:
=================

- start-simple.ps1 (script simples sem caracteres especiais)
- start-all.ps1 (script completo)
- index.html (interface visual)
- start-html.bat (script alternativo)
- INSTRUCOES-TERMINAL.md (documentacao completa)
- COMO-USAR.txt (este arquivo)

REQUISITOS:
===========

- Windows 10/11
- .NET 8.0 ou superior
- Python 3.7 ou superior
- PowerShell (vem com Windows)
- Counter-Strike 2 com GSI configurado
